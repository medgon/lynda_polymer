Two-way Data Binding in Polymer

Two-way data binding in Polymer is declared with curly braces: {{prop-name}}

Data flows from the point where it is changed to the target. For example, in this case if the prop-name property is changed on the model, then the target element where the braces are defined gets the new data.

If the data changes on the target element itself, then the data flows back up to the model.
